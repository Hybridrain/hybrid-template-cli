Node version 10.15.3 or higher recommended!

Check Version
```
--version
```
```
-V
```

Standard
```
hr-template create template
```
Standard shortcut
```
hr-template c template
```
No bullshit
```
--nbs
```
Vue

```
--vue
```
SCSS
```
--scss
```
Sass
```
--sass
```
